# BIVA Auto-Detector

Aplicação web estática para auxiliar a leitura de gráficos BIVA:

- auto-detecção da área dos eixos;
- busca de múltiplas hipóteses de elipse na imagem;
- ajuste manual por alças no canvas;
- cálculo de rho por ângulo da elipse e, opcionalmente, por desvio-padrão do ângulo de fase;
- ponderação por N para melhorar a estimativa quando mais dados do grupo são fornecidos.

## Uso

Abra `index.html` no navegador ou publique a raiz do repositório via GitHub Pages.
